Popis funkce

Makro projde všechny listy v aktuálním sešitu (ThisWorkbook) a postupně zapíše jejich názvy do sloupce A aktivního listu, počínaje buňkou A1.

Vstup

Program nevyžaduje žádný vstup od uživatele.

Výstup

Do sloupce A aktivního listu se vypíší názvy všech listů.
Po dokončení se zobrazí informační hlášení.
